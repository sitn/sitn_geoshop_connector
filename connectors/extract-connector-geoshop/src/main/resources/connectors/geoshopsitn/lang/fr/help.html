Le connecteur geoshop permet d'importer des commandes depuis un serveur geoshop et d'y exporter les résultats.

Configuration requise :

  • URL du service : L'URL de l'api du geoshop
  • Login distant : Identifiant de connexion au serveur exclusif à Extract
  • Mot de passe : Mot de passe de connexion
  • Taille maximale d'upload (Mo) : Limite de taille pour l'export des fichiers
  • URL de détail de commande : Pattern d'URL pour accéder aux détails d'une commande
